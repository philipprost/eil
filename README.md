# EIL - Emoji Interaction Library

Convert text emojis into their actual emoji representations with ease!

## Installation

**npm**:

```bash
npm install eil-js
```

**cdn**:

`<script src="https://unpkg.com/eil-js@0.0.1/dist/emojiInteraction.js"></script>`
